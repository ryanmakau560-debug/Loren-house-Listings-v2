# Loren House Listings

## Overview

**Loren House Listings** is a simple real estate listing website designed to connect **property buyers directly with real estate companies and property sellers**.

Instead of complicated sign-ups or messaging systems, the platform allows buyers to quickly view available properties and **contact the seller directly through their phone number**.

The goal of the project is to make property searching **simple, fast, and direct**, especially for users who prefer immediate communication.

---

## Features

* View property listings in a simple and clean interface
* Each listing includes a **phone number for direct contact with the seller**
* Designed to reduce middlemen between buyers and sellers
* Lightweight and fast-loading website
* Simple user interface that focuses on the listings

---

## Technologies Used

This project was built using:

* **JavaScript** – for interactive functionality
* **Tailwind CSS** – for styling and layout
* **HTML** – for the structure of the website

The use of **Tailwind CSS** helps keep the design clean while allowing fast development.

---

## Purpose of the Project

This project was created as a **learning and practice project** to demonstrate basic web development skills and how simple technologies can be used to build a functional website.

It focuses on:

* Frontend layout
* UI design
* Basic website structure
* Connecting users with real-world services

---

## How It Works

1. Users open the website homepage.
2. They browse through available house listings.
3. When they find a property they like, they can **call the seller directly using the provided phone number**.

This keeps communication **simple and immediate**.
